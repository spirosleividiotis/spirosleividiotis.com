Production Pipeline

Lotties are used everywhere at Qonto: in-product micro-interactions, website illustrations, onboarding flows, empty states, success feedback. The library grew to 1,000+ files stored in LottieFiles, covering every use case from button states to full-screen celebrations.

In the past 6 months, I led the migration from .json to .lottie format—a smaller, more efficient file type that reduced bundle sizes on Android. I also established technical guidelines around format limitations, compression, and cross-platform compatibility, becoming the go-to for pipeline issues and technical questions.

Beyond Individual Files

The library became a production asset. Engineers integrated Lotties confidently. Designers referenced existing work before creating new animations.
