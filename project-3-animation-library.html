Creating at Scale

I designed and produced 1,000+ Lottie animations used across Qonto: in-product micro-interactions, website illustrations, onboarding flows, empty states, success feedback. Every animation was optimized for performance, tested across platforms, and ready to ship.

Beyond production, I built the pipeline: established the LottieFiles library structure, defined export standards, and over the past 6 months led the .json to .lottie migration—reducing file sizes and improving Android performance. I became the technical owner: format guidelines, compression, cross-platform compatibility, troubleshooting for the team.
